# 아두이노 수치 해석을 활용한 밀폐 생태계 내 탄소 순환의 평형 조건

### 사전 조사
**가. 식물의 기체 대사**
- 낮과 밤의 공기 변화: 낮에는 광합성으로 이산화탄소를 흡수하고 산소를 내뱉고 밤에는 호흡으로 산소를 쓰고 이산화탄소를 내뿜음
- 광보상점: 빛의 밝기에 따라 식물의 흡수하는 이산화탄소와 뱉는 이산화탄소의 양이 같아져, 겉보기에 기체 교환이 0이 되는 지점(200~500)
- 광포화점: 식물이 받을 수 있는 빛의 최대의 세기로 이 이상으로 빛의 세기가 높아져도 광합성량이 늘지 않는 지점(2000ux)
  이산화탄소 농도: 대기 중 0.1%에서 가장 많음==포화점
- 이끼 조사: 좁은 밀폐 공간에 오래 버티는 비단이끼의 특성과 시간당 평균 기체 흡수, 배출량을 조사

**나. 닫힌 공간과 기체 법칙**
- 이상기체 상태방정식(PV=nRT): 상자 안 기체의 온도, 압력, 부피, 분자의 개수 사이의 유기적 관계 분석(p==압력,v==부피,t==절대 온도,R==기체상수,n==기체 mol수, R==0.082)
- 1기압, 17,253.04==부피, t==273도, n==700mol
- 기체 분자의 운동: 조명 열로 인해 내부 온도가 올라갈 때 기체 알갱이들의 움직입과 압력이 변하는 원리
- 에너지보존: 공기는 통하지 않고 빛과 열만 드나드는 '닫힌 공간'에서 내부의 전체 에너지 총량이 일정하게 보존되는 법칙
**A. 광합성 량**
- 온도 방정식:10도 상승=>광합성 속도 2배 빨라짐(q10법칙)
- 
- p(광합성 속도)=(pmax(최대 광합성 속도)*I(빛의 세기)/(k(광합성 최대 속도/2)+I)
**다. 아두이노 정밀 측정**
- 기체 센서 활용: 대기 중 농도(약 400~500ppm)부터 주사기 주입 후 수만 ppm까지 측정 가능한 이산화탄소 센서와 온도에 따른 식물의 광합성량을 관측하기 위해 온습도 센서 준비
- 데이터 로깅: 센서가 PC실시간(10분 간격)으로 자동 저장하는 코드를 준비
- 완벽 밀봉: 센서 선이 상자 밖으로 나올 때 공기가 새지 않도록 고무 마개, 글루건 등을 사용해 틈새를 막음
